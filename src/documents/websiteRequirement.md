We need a custom, professional liquor store e-commerce website for **Hamilton Liquor Store** . 

**Business Name:** Hamilton Liquor Store 

**Business Type:** Liquor store 

**Address:** 5418 Harford Rd, Baltimore, MD 21214 

**Phone:** (443) 438-5483 

## **Payment Processor:** SpotOn 

**Preferred Website Platform:** Custom Next.js / React website 

**Main Goal:** Build a modern online liquor store website where customers can browse products, view specials, place pickup orders, request delivery if legally approved, and pay online if SpotOn integration supports it.

The website should be fast, mobile-friendly, professional, easy for customers to use, and easy for store staff to manage.

## 10. Pickup and Delivery Requirements 

The website should support in-store pickup first. 

Delivery should be built as an optional feature that can be turned on only after Hamilton Liquor Store confirms that delivery is legally allowed and properly permitted. 

Pickup requirements: 

- Customer selects pickup time 

- Customer receives confirmation 

- Store prepares the order 

- Customer must show valid government-issued photo ID 

- Customer must be 21 or older 

- Store can refuse pickup if ID is invalid 

Delivery requirements, if enabled: 

- Delivery radius ( 2 – 2.5 miles) or zip code settings 

- Delivery fee $4.99 if more than $200 dollar order delivery is free 

- Minimum order amount ( $30 ) 

- Delivery schedule 

- Store-controlled delivery hours 

- Driver/order handoff instructions 

- Valid ID required at delivery 

- Customer must be 21 or older 

- Alcohol cannot be left unattended 

- Delivery may be refused if ID is invalid 

- Delivery may be refused if the customer appears intoxicated 

- Delivery should only happen during legally allowed store operating hours 

11. Age Verification and Alcohol Compliance 

Because this website sells alcohol, age verification is very important. 

The website must include: 

- 21+ age verification popup before entering the website 

- Date of birth field at checkout 

- 21+ confirmation checkbox at checkout 

- Notice that valid government-issued photo ID is required for pickup or delivery 

- Notice that alcohol cannot be sold to anyone under 21 

- Notice that the store may refuse or cancel orders if age cannot be verified 

- Age Verification Policy page 

- Pickup and Delivery Policy page 

- Refund Policy page 

- Fulfillment Policy page 

The developer should not provide legal advice. Hamilton Liquor Store should confirm final alcohol e-commerce, pickup, and delivery rules with the proper Maryland/Baltimore licensing authority before launch. 

## 12. SpotOn Payment Integration 

Hamilton Liquor Store currently uses SpotOn for credit card processing in-store. 

Before building the final checkout system, the developer must verify SpotOn integration options. 

The developer should confirm: 

- Does this SpotOn merchant account support online payments? 

- Does SpotOn provide API access for custom websites? 

- Does SpotOn offer hosted checkout? 

- Can SpotOn process alcohol e-commerce payments? 

- Can online orders sync with SpotOn POS? 

- Can product inventory sync with SpotOn? 

- Can taxes, fees, refunds, and receipts sync with SpotOn? 

- Does SpotOn support Apple Pay and Google Pay online? 

- Does SpotOn support card-not-present payments for this business type? 

Preferred option: 

Use SpotOn for online checkout if the account supports it. 

Backup options if SpotOn does not support custom website checkout: 

- Square, only if alcohol sales are approved 

- Stripe, only if alcohol sales are approved 

- Another alcohol-friendly payment gateway 

- Manual payment at pickup as a temporary option 

Please provide two estimates: 

1. Website with SpotOn integration 

2. Website without SpotOn integration using a backup payment method
